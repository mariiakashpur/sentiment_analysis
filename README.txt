Sentiment Analysis Mini-Projects 1 and 2
Author: Mariia Kashpur

For mini-project 2 results, run "polarity.py". Command line arguments to be passed: positive words dataset, negative words dataset, a reviews corpus.
The words "excellent" and "poor" are used to create a lexicon, and the frame is 5 words. 

For mini-project 1 results, run "classify.py". Command line arguments to be passed: path to polarity lexicon pickle file from project 2, positive reviews dataset, negative reviews dataset.
